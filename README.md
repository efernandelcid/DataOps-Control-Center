# DataOps Control Center

Sistema web para monitoreo de bases de datos PostgreSQL, SQL Server y Oracle.

## Funcionalidades principales

- Registro de conexiones de bases de datos.
- Monitoreo automático cada 30 segundos.
- Validación de estado ACTIVE / ERROR.
- Historial de revisiones.
- Métricas avanzadas.
- Alertas críticas.
- Gráfica de tiempos de respuesta.
- Exportación de reporte CSV.

## Tecnologías utilizadas

- Node.js
- Express
- PostgreSQL
- SQL Server
- Oracle Database
- Docker
- HTML, CSS y JavaScript
- Chart.js

## Módulos del sistema

- Backend API REST
- Dashboard web
- Cron Job de monitoreo
- Registro histórico de métricas
- Exportación de reportes
